# About

ng-openapi is an open-source Angular-first OpenAPI client generator, released under the [MIT license](https://github.com/ng-openapi/ng-openapi/blob/main/LICENSE).

It is created and maintained by [Tareq Jami](https://tareqjami.de). If you'd like to support the project, you can [sponsor it on GitHub](https://github.com/sponsors/ng-openapi).
